# StackChan 功能列表

> 基于开源仓库四个模块（app / firmware / server / remote）的源码分析，按模块和功能域梳理。

## 📟 一、机器人固件（firmware/）— 运行于 ESP32-S3 / CoreS3

### 🔹 AI Agent 智能体
- [x] **语音唤醒** — 唤醒词 "Hi, StackChan"（基于 WN9 唤醒词模型）
- [x] **点击唤醒** — 单击触摸屏唤醒语音交互
- [x] **LLM 对话** — 接入小智 AI 平台，支持多模型切换
- [x] **语音合成 (TTS)** — AI 语音回复，支持音色、语速、声调调节
- [x] **语音识别 (ASR)** — 用户语音输入识别，支持识别速度调节
- [x] **LED 状态指示** — 绿灯（聆听）、蓝灯（讲话）、不亮（闲置）
- [x] **语音指令控制** — 通过语音控制设备自身部件（音量、转头、灯光、摄像头查看、电量查询）
- [x] **Home Assistant MCP 集成** — 通过 MCP 连接 HA，语音控制智能家居设备
- [x] **联网 OTA 固件更新** — 每次进入 AI Agent 自动检查并更新固件

### 🔹 Avatar 面部表情系统
- [x] **基础表情库** — 中性、快乐、悲伤、愤怒、惊讶、眨眼等
- [x] **表情切换** — 支持实时切换面部表情状态
- [x] **眼睛/嘴部控制** — 可独立控制左眼、右眼、嘴部的位置和大小
- [x] **气泡文本** — 在屏幕上显示文字气泡
- [x] **装饰器 (Decorator)** — 瞬态叠加视觉效果：爱心❤️、冷汗💧、眩晕😵‍💫、害羞🥰、愤怒😠
- [x] **皮肤机制 (Skin)** — 可更换的面部皮肤渲染（默认皮肤、表情动画风格、微信风格等）

### 🔹 动画系统
- [x] **关键帧动画** — 基于 KeyframeSequence 定义面部特征和舵机角度的关键帧
- [x] **时间线播放器 (Timeline)** — 支持循环播放、暂停、恢复
- [x] **JSON 远程加载** — 动画数据可通过 JSON 远程下发

### 🔹 行为修饰器 (Modifiers)
- [x] **自动眨眼** — 周期性无意识眨眼
- [x] **呼吸效果** — 身体/面部微小的呼吸起伏
- [x] **说话嘴型同步** — 说话时嘴部自然张合
- [x] **摸头反应** — 触摸顶部感应区触发特定表情和动作
- [x] **空闲随机动作** — 闲置时随机转动头部
- [x] **空闲随机表情** — 闲置时切换表情
- [x] **IMU 姿态反应** — 摇晃整机触发头晕😵‍💫表情
- [x] **睡眠表情** — 长时间无交互后自动进入睡眠😴状态
- [x] **跳舞动画** — 配合音乐编排的动作序列

### 🔹 运动控制系统
- [x] **水平舵机 (Yaw)** — 360° 无限旋转（水平摇头）
- [x] **垂直舵机 (Pitch)** — 90° 运动范围（上下点头）
- [x] **舵机反馈** — 实时读取舵机当前位置和状态
- [x] **位置控制** — `moveYaw()` / `movePitch()` / `move()` 精确控制
- [x] **归位功能** — `goHome()` 回到初始位置
- [x] **逆运动学** — `lookAtNormalized()` / `lookAtPoint()` 看向指定坐标
- [x] **舵机校准** — 设置中提供校准流程，手动调整初始位置
- [x] **运动锁定** — `setModifyLock()` 防止修饰器干扰

### 🔹 无线通信
- [x] **WebSocket 云端连接** — 连接后端服务器，支持双向实时通信
- [x] **BLE 蓝牙外设** — 基于 NimBLE 栈，用于配网和本地控制
- [x] **ESP-NOW 接收** — 接收遥控器广播，低延迟控制舵机
- [x] **WiFi Station 模式** — 连接 2.4GHz 无线路由器
- [x] **声学配网** — 通过声音信号配置 WiFi

### 🔹 硬件驱动
- [x] **2.0" IPS 触摸屏** — 320×240，ILI9342C 驱动，FT6336U 电容触摸
- [x] **GC0308 摄像头** — 640×480, 0.3MP，支持 JPEG 编码推流
- [x] **双麦克风 + ES7210 音频编码** — 语音采集
- [x] **1W 扬声器 + AW88298 I2S 功放** — 语音播放
- [x] **九轴 IMU** — BMI270 (加速度+陀螺仪) + BMM150 (地磁)
- [x] **12 颗 RGB LED (WS2812C)** — 两排装饰灯
- [x] **三区触控板 (Si12T)** — 顶部触摸感应
- [x] **全功能 NFC (ST25R3916)** — 近场通信
- [x] **红外发射/接收 (IRM56384)** — 红外遥控
- [x] **550mAh 电池 + AXP2101 电源管理** — 充放电管理
- [x] **RTC (BM8563)** — 实时时钟
- [x] **microSD 卡槽** — 扩展存储
- [x] **Grove 接口 × 3** — 连接 M5Stack 生态扩展单元

## 🖥️ 三、后端服务器（server/）— Go + GoFrame v2

- [x] **26 种消息类型** (0x01~0x1A) — Opus 音频、JPEG 视频、控制命令、通话信令、心跳等
- [x] **双连接池管理** — `stackChanClientPool`（硬件）+ `appClientPool`（App）
- [x] **双向中继** — App ↔ 服务器 ↔ 机器人之间的任意方向消息转发
- [x] **音频流中继** — Opus 编码的实时语音数据中转

- [x] **通话信令** — 呼叫请求、接听、拒接、挂断
- [x] **心跳保活** — 每 5 秒 Ping/Pong，60 秒超时断开

| **全景图 (v1)** | 全景数据 CRUD |

| **应用商店 (v1)** | 应用列表查询 |

| **管理后台 (admin)** | 管理员登录、应用 CRUD、文件管理 |

## 🎮 四、遥控器固件（remote/）— ESP-IDF (ESP32 / StickC-Plus)

### 🔹 三模式操作
- [x] **Mode 0: SETUP（设置模式）** — 配置 WiFi 频道 (1~14) 和目标接收器 ID (0~50)
- [x] **Mode 1: RUNNING（摇杆遥控）** — 通过摇杆控制机器人舵机
- [x] **Mode 2: IMU（体感遥控）** — 通过倾斜遥控器控制机器人

### 🔹 摇杆控制
- [x] **原始值读取** — I2C 读取 Hat Mini JoyC 摇杆 XY 值
- [x] **死区过滤** — 中心区 ±300 的死区，防止抖动误触
- [x] **变化量滤波** — 变化阈值 < 5 时不上报，避免网络拥塞
- [x] **摇杆→舵机映射** — XY 原始值映射到 Yaw (-1280~1280) 和 Pitch (0~900)

### 🔹 IMU 体感控制
- [x] **加速度计数据读取** — 读取 StickC-Plus 内置 IMU
- [x] **俯仰/滚转角计算** — 基于加速度计计算设备姿态
- [x] **角度限幅** — 滚转 [-1.5, 1.5]，俯仰 [0, 1.5]
- [x] **姿态映射到舵机** — 倾斜角度转换为舵机控制指令

### 🔹 ESP-NOW 通信
- [x] **低延迟广播** — 基于 WiFi STA 模式的 ESP-NOW 点对点协议
- [x] **8 字节数据包** — 目标 ID(1) + Yaw(2) + Pitch(2) + Speed(2) + BtnB(1)
- [x] **频道切换** — 支持 1~13 频道，同频道设备才能通信
- [x] **目标 ID 寻址** — 0=广播所有设备，非 0=特定设备
- [x] **ESP-NOW 重初始化** — 模式切换时重新配置

### 🔹 LCD 用户界面 (LVGL v8.4)
- [x] **设置屏幕** — 频道和 ID 的数值选择器，摇杆上下滚动，BtnB 切换选项
- [x] **运行屏幕** — 摇杆位置可视化（115×115 十字准星 + 红点）、电池百分比、频道/ID 显示
- [x] **IMU 屏幕** — 3D 线框立方体实时显示设备姿态（12 条边，正交投影，旋转矩阵计算）

### 🔹 硬件功能
- [x] **按钮事件** — BtnA 循环切换三种模式、BtnB 切换选中项
- [x] **电池电量监测** — 通过 AXP192 PMIC 读取电池百分比
- [x] **IMU 传感器** — SH200Q 六轴传感器
- [x] **I2C 通信** — 100kHz 与 Hat Mini JoyC 通信
- [x] **NVS 持久化** — 存储 ESP-NOW 配置参数

## 🔄 五、跨模块协作功能

| 功能 | 涉及模块 | 通信方式 | 说明 |
|------|---------|---------|------|
| **AI 对话** | firmware + app + server + 小智平台 | WebSocket + HTTP | 语音采集→识别→LLM→TTS→播放，全链路 |
| **视频监控** | firmware + app + server | WebSocket (JPEG 流) | 摄像头画面实时传输到手机 |
| **远程控制** | app + server + firmware | WebSocket (二进制协议) | 手机控制机器人舵机和表情 |
| **舞蹈编排** | app + firmware | BLE / WebSocket | 编辑舞蹈→发送到机器人播放 |
| **本地遥控** | remote + firmware | ESP-NOW (8B 广播) | 摇杆/IMU 直接控制，不经过服务器 |
| **OTA 更新** | firmware + server | HTTP | 检查→下载→安装新固件 |
| **设备配网** | app + firmware | BLE GATT | 通过蓝牙配置 WiFi |
| **MCP 集成** | firmware + app + HA 插件 | WebSocket | 语音控制智能家居设备 |
